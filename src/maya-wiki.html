What is Maya?

Maya is a standalone package manager written in bash, which follows the same scheme as ports, for example.
It uses pre-built recipes to download, check dependencies, compile and install binaries
And, if removal is necessary, it also removes the binaries and all traces they leave behind.

Below you will find an example of installation and recipe.
   $ maya --about make
     make :: 4.4.1             GNU make source code.
   $ maya --install make
     # [the entire compilation and installation process...]
And removing it becomes equally simple:
   $ maya --remove make
And below is an example recipe:
   $ cat ~/.local/share/maya/pkg/make 
     
     #!/bin/bash

     NAME="make"
     VERSION="4.4.1"
     DESCRIPTION="GNU make source code."
     SOURCE="https://ftp.gnu.org/gnu/$NAME/$NAME-$VERSION.tar.gz"
     DEPENDENCIES=(
         "wget"
         "tar"
         "gcc"
     )

     function install(){
         wget -c $SOURCE --directory-prefix=/tmp/maya/$NAME
         tar -xvf /tmp/maya/$NAME/$NAME-$VERSION.tar.gz --directory=/tmp/maya/$NAME
         (
             cd /tmp/maya/$NAME/$NAME-$VERSION
             sudo ./configure --disable-dependency-tracking
             sudo ./build.sh
             sudo ./make
             sudo ./make install
         )
     }

     function remove(){
         LOCALE="/usr/local/share/locale"
         sudo rm -f /usr/local/bin/make
         sudo rm -f /usr/local/include/gnumake.h
         sudo rm -f /usr/local/share/man/man1/make.1
         sudo rm -f /usr/local/share/info/{make.info,make.info-1,make.info-2,make.info-3}
         sudo rm -rf $LOCALE/be/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/bg/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/cs/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/da/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/de/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/es/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/fi/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/fr/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/ga/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/gl/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/he/LC_MESSAGES/make.mo
         sudo rm -rf $LOCALE/hr/LC_MESSAGES/make.mo

This means that ANYONE can contribute if they want, without actually needing a ready-made repository, as long as everything is filled in correctly.
If you are interested, there is a package-template.sh file in ~/.local/share/maya/src/package-template.sh
